Web Development
w3schools.com W3 Schools is a great reference site for any web developer. Whenever I'm wondering about a certain CSS property, or correct JavaScript syntax, that's the first place I go.
html5boilerplate.com HTML 5 Boilerplate is a "Rock solid default template for HTML 5." Basically that means that it's a CSS stylesheet written by web guru Paul Irish that you can use as a building block when developing a website. I use it on almost every site I build.
fontsquirrel.com Font Squirrel is a great site for web developers who want to use non–standard fonts on their websites as text rather than images. Once you have selected your font on Font Squirrel, you can download it along with @font–face CSS code.
Nettuts+ Nettuts+ is an awesome web design and development tutorial website. There are a wide variety of tutorials available in any web development discipline and the prices are very reasonable.
jquery.com The official jQuery website comes in handy whenever I'm developing a site with this technology. The ability to find descriptions about all things jQuery and find properly written code to use for reference is extremely helpful, and saves time.
php.net This is the official website for PHP. Same concept here as the jQuery website. This site is my main reference for any PHP project.
Computer Programming
java.about.com I'm a big fan of About.com. I'm constantly amazed at how many different topics they have articles and tutorials on. In this case, the topic of choice is Java programming, which is something that I'll be studying extensively at CTU. This is a great place to get started.
Oracle.com 'New To Java' Oracle's 'New to Java' site is a great reference tool for anyone looking to get started programming in Java.
BigNerdRanch.com The Big Nerd Ranch was created by Aaron Hillegass, one of the original programmers at neXtStep with Steve Jobs. Mr. Hillegass later worked for Mr. Jobs at Apple Computers as a programming instructor. After many years at neXtStep and Apple, Aaron branched off on his own to create the Big Nerd Ranch, a sanctuary of sorts for programmers of all levels who want to come for a stay at the ranch in Atlanta and study programming in a variety of languages with some of the nation's best instructors. The Big Nerd Ranch is also responsible for a number of quality programming books on topics such as: IOS Development, Cocoa programming, Android Development, and much more.
Cocoa Is My Girlfriend This is a nice reference site for anyone interested in cocoa programming for the macintosh platform.
codingbat.com Coding bat is a great website for Java or Python programmers. The site allows you to create a profile and complete programming challenges that get progressively more difficult. An account is not necessary to complete challenges, but it is necessary if you want to save your progress and resume in the future.
Design
kuler.adobe.com Kuler is a site by Adobe that allows users to submit color palettes. You can browser the most popular palettes or search for a specific one. I find this site very useful when initially designing a site.
2advanced.com 2 Advanced Studios was started by Eric Jordan in the late 90's, and quickly gained a reputation as one of the most advanced Flash animation and development companies in America. The work done by Eric Jordan and his team was a huge inspiration to me during my formative years as a web developer. Although my appreciation of Flash has decreased over the years, I'm still a huge fan of 2 Advanced Studios and always look forward to viewing their latest projects.
Chapter3.net It's hard for me to mention 2 Advanced Studios without mentioning Chapter3.net and Mr. Jens Karlson. I've been a fan of Chapter3.net just as long as 2 Advanced Studios and even have one of Jens Karlson's designs tattooed on my back. His style is futuristic and technical with strong use of light and three dimensional spaces.
TemplateMonster.com TemplateMonster.com is a great resource for website and logo designers. They offer a huge assortment of logo and website templates for sale. Whether you're looking to save time or you need design inspiration, this is a great site to check out.
